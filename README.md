## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/zhwltlr/js_example

# Access
$ cd js_example

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
